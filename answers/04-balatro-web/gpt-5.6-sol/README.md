# Joker's Orbit · 小丑星轨

一个原创的单页 Roguelike 扑克构筑游戏学习项目。无需安装依赖或联网。

## 运行

直接双击 `index.html`，或在本目录运行：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000`。

## 操作

- 点击卡牌：选择或取消选择，最多 5 张。
- 打出选牌：按牌型、小丑牌效果逐项结算。
- 弃掉选牌：消耗一次弃牌并补至 8 张。
- `Space`：打出选牌；`D`：弃掉选牌。
- 击破盲注后可在商店购买小丑牌，最多持有 5 张。

游戏包含 9 个盲注、9 类原创小丑牌、完整牌型识别、动态难度曲线、商店与胜负循环。

## 文件

- `index.html`：完整游戏代码（HTML + CSS + JavaScript）。
- `assets/cosmic-jester-bg.png`：原创主视觉背景。

本项目使用原创名称、代码与视觉资产，仅用于学习与演示。
