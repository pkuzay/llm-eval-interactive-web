飞行系统

本模拟器使用三维旋转群 SO(3) 与李代数 so(3) 的指数映射进行精确姿态积分（Rodrigues 公式 R(t+Δt) = R(t)·exp(ω̂·Δt)），彻底避免欧拉角万向节死锁。耦合欧拉方程完整建模陀螺力矩 I·ω̇ = τ − ω×(Iω)。Acro 模式下摇杆输入直接映射为目标角速度，三轴 PID 闭环输出力矩。

遥控器接入

支持 Web Gamepad API（OpenTX/EdgeTX 遥控器 USB HID 模式）。将遥控器通过 USB 连接至电脑，浏览器识别为 Gamepad 设备即可。默认通道映射：Axes[0]=Roll, [1]=Pitch, [2]=Throttle, [3]=Yaw。若设备映射不同，请参考 HUD 左下角的实时 AXIS 监测值自行调换。

无遥控器时可使用键盘：WASD Pitch/Roll，QE Yaw，↑↓ 油门。

VTX 图传衰弱

5.8GHz 模拟图传效果根据距离起飞原点衰减：信号弱时出现雪花纹与色彩饱和度降低；接近极限距离时画面横向撕裂、滚动黑条、色度分离；超出范围将完全失联，无人机失去推力坠毁。

⏳ 等待遥控器连接（也可使用键盘）
