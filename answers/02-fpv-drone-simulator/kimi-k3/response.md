## 核心实现要点

**1. SO(3) 姿态积分（李群/李代数）**
- 姿态 `R ∈ SO(3)` 用单位四元数存储（SO(3) 双覆盖，彻底无万向节死锁）
- 指数映射闭式解：θ = |ω|Δt，Δq = (ω̂·sin(θ/2), cos(θ/2))，右乘 `q ← q ⊗ Δq`（机体系增量），每步归一化维持流形约束
- 物理采用 **240Hz 固定步长子步积分**，与渲染帧率完全解耦，保证数值精度

**2. 刚体动力学（手写，零第三方物理引擎）**
- 转动：`I·ω̇ = τ − ω×(Iω) − c·ω`，陀螺项 `ω×Iω` 完整保留（翻滚时俯仰/偏航耦合的关键）
- 平动：`m·v̇ = T·(R·ŷ) + m·g − k|v|v`，推力沿机体轴、二次方空气阻尼
- 参数对标 5 寸花飞机：0.72kg，悬停油门约 30%

**3. Acro 模式 + 三轴 PID**
- 摇杆经死区 + 指数曲线（Expo 0.35）映射为**目标角速度**（横滚/俯仰 7.2 rad/s，偏航 5.2 rad/s）
- PID 含积分抗饱和钳制 + D 项一阶低通滤波，输出力矩直接驱动刚体

**4. USB HID 遥控器接入**
- Web Gamepad API 自动识别 OpenTX/EdgeTX 设备（按任意摇杆键激活），AETR Mode 2 通道映射，油门 −1..1 → 0..1，HUD 实时显示四通道数值；无遥控器时自动回退键盘

**5. VTX 图传衰弱着色器（GLSL 全屏后处理）**
- 信号按距离幂律衰减（400m 极限），分阶段退化：去饱和 → 雪花白噪声 → 横向撕裂（扫描行随机错位 + 长撕裂带）→ 帧垂直跳动 → 黑色滚动条纹 → 彻底失联纯雪花
- 失联即切断动力坠机（Failsafe），另有硬着陆炸机判定，按 R 复位

**场地**：6 个高对比度发光环形门（错落高度/朝向）、3 道带钻越窗的废墟墙、5 根绕桩立柱、远处归航地标塔、程序化草地纹理 400× 平铺、渐变天穹 + 太阳盘 + 软阴影。

**操作**：遥控器即插即飞；键盘 ↑↓←→ 俯仰/横滚、A/D 偏航、W/S 油门、R 复位、C 切换追尾视角（可见螺旋桨随油门旋转的机模）。
